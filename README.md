# PERCOBAAN 1: PENGENALAN SISTEM TERTANAM

## Percobaan 1: Blink & Buzz - Pengenalan Output Digital

### Deskripsi
Percobaan ini merupakan pengenalan dasar penggunaan output digital pada sistem tertanam. Program mengontrol dua komponen output secara bersamaan:
- **LED** untuk output visual
- **Buzzer** untuk output audio

### Cara Kerja
1. LED dan buzzer menyala bersamaan selama 1 detik
2. Kemudian keduanya mati selama 1 detik
3. Proses ini berulang secara terus-menerus

### Komponen Utama
- **LED** pada pin 8
- **Buzzer** pada pin 7

### Tujuan Pembelajaran
Memahami konsep dasar penggunaan `digitalWrite()` dan `tone()` untuk mengontrol output digital pada mikrokontroler Arduino.
